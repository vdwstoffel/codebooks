# Introduction to JavaScript

<div style={{display: 'flex', justifyContent: 'center', alignItems: 'center'}}>

JavaScript is a versatile, high-level, and interpreted programming language primarily known for enabling interactive web pages. Alongside HTML and CSS, it is one of the three core technologies of the World Wide Web. It can also be used for server-side development (Node.js), mobile apps, and desktop applications.

## Running JavaScript

### In a Web Browser

You can embed JavaScript directly into an HTML file using `<script>` tags.

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First JS Page</title>
</head>
<body>
    <script>
        console.log("Hello from the browser console!");
        alert("Hello from an alert box!");
    </script>
</body>
</html>
```

### Using Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows you to run JavaScript code outside of a web browser, typically for server-side applications or command-line tools.

1.  **Create a file** (e.g., `hello.js`):

    ```javascript
    console.log("Hello from Node.js!");
    ```

2.  **Run from your terminal**:

    ```bash
    node hello.js
    ```
