# Strings in JavaScript

Strings are a fundamental data type in JavaScript used to represent textual data. They are sequences of characters enclosed in single quotes (`'`), double quotes (`"`), or backticks (`` ` ``).

## String Immutability

In JavaScript, strings are **immutable**. This means that once a string is created, its content cannot be changed. Any operation that appears to modify a string (like `replace()` or `toUpperCase()`) actually returns a *new* string, leaving the original string unchanged.

```javascript
let originalString = "Hello";
let modifiedString = originalString.toUpperCase();

console.log(originalString); // "Hello" (original remains unchanged)
console.log(modifiedString); // "HELLO" (a new string is returned)
```

## Creating Strings

### String Literals

The most common way to create strings.

```javascript
let singleQuotes = 'Hello, World!';
let doubleQuotes = "Hello, JavaScript!";
```

### Template Literals (Template Strings)

Introduced in ES6, template literals use backticks (`` ` ``) and offer enhanced functionality like multi-line strings and string interpolation.

```javascript
let name = "Alice";
let greeting = `Hello, ${name}!`; // "Hello, Alice!"

let multiLine = `This is a
multi-line
string.`;
console.log(multiLine);
// Output:
// This is a
// multi-line
// string.
```

### Converting to String

You can convert other data types to strings.

```javascript
let num = 123;
let str1 = num.toString();    // "123"
let str2 = String(num);       // "123"
let str3 = num + "";          // "123" (type coercion)
```

## Basic Properties and Access

### Length

The `length` property returns the number of characters in a string.

```javascript
let text = "JavaScript";
console.log(text.length); // 10
```

### Accessing Characters

You can access individual characters using bracket notation (like arrays) or the `charAt()` method.

```javascript
let word = "Hello";
console.log(word[0]);      // "H"
console.log(word.charAt(1)); // "e"
```

## String Manipulation

### Concatenation

Joining strings together.

```javascript
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; // "John Doe"
let combined = "Hello ".concat("World");   // "Hello World"
```

### Substrings

Extracting parts of a string.

-   `slice(startIndex, endIndex)`: Extracts a part of a string and returns the extracted part in a new string. `endIndex` is optional and not included.
-   `substring(startIndex, endIndex)`: Similar to `slice()`, but handles arguments differently (e.g., negative values are treated as 0).

```javascript
let sentence = "The quick brown fox";

let part1 = sentence.slice(4, 9);    // "quick"
let part2 = sentence.substring(10);  // "brown fox"
```

### Replacing Content

-   `replace(searchValue, replaceValue)`: Replaces the *first* occurrence of a substring.
-   `replaceAll(searchValue, replaceValue)`: Replaces *all* occurrences of a substring (ES12+).

```javascript
let text = "Hello World, Hello Universe!";

let newText1 = text.replace("Hello", "Hi"); // "Hi World, Hello Universe!"
let newText2 = text.replaceAll("Hello", "Hi"); // "Hi World, Hi Universe!"

// Using regular expressions for global replacement (older way for replaceAll)
let newText3 = text.replace(/Hello/g, "Hi"); // "Hi World, Hi Universe!"
```

### Splitting and Joining

-   `split(separator)`: Splits a string into an array of substrings.
-   `join(separator)`: (Array method) Joins elements of an array into a string.

```javascript
let csv = "apple,banana,orange";
let fruitsArray = csv.split(","); // ["apple", "banana", "orange"]

let words = ["JavaScript", "is", "fun"];
let sentence = words.join(" "); // "JavaScript is fun"
```

### Trimming Whitespace

`trim()` removes whitespace from both ends of a string.

```javascript
let padded = "   Hello, World!   ";
let trimmed = padded.trim(); // "Hello, World!"
```

## String Searching and Comparison

### Checking for Substrings

-   `includes(substring)`: Checks if a string contains a specified substring (case-sensitive).
-   `startsWith(prefix)`: Checks if a string begins with a specified prefix.
-   `endsWith(suffix)`: Checks if a string ends with a specified suffix.

```javascript
let mainString = "Learning JavaScript";

console.log(mainString.includes("Java"));    // true
console.log(mainString.startsWith("Learn")); // true
console.log(mainString.endsWith("Script"));  // true

// Case-insensitive check
console.log(mainString.toLowerCase().includes("java")); // true
```

### Finding Substring Position

-   `indexOf(substring)`: Returns the index of the first occurrence of a substring.
-   `lastIndexOf(substring)`: Returns the index of the last occurrence of a substring.

```javascript
let data = "apple,banana,apple,orange";

console.log(data.indexOf("apple"));     // 0
console.log(data.lastIndexOf("apple"));  // 13
console.log(data.indexOf("grape"));     // -1 (not found)
```

### Case-Insensitive Comparison

Convert both strings to the same case before comparing.

```javascript
let strA = "Hello";
let strB = "hello";

if (strA.toLowerCase() === strB.toLowerCase()) {
    console.log("Strings are equal (case-insensitive)"); // "Strings are equal (case-insensitive)"
}
```

## Case Transformation

-   `toUpperCase()`: Converts a string to uppercase.
-   `toLowerCase()`: Converts a string to lowercase.

```javascript
let mixedCase = "JavaScript";

console.log(mixedCase.toUpperCase()); // "JAVASCRIPT"
console.log(mixedCase.toLowerCase()); // "javascript"
```

### Capitalize First Letter

```javascript
let sentence = "first letter capitalized";
let capitalizedSentence = sentence.charAt(0).toUpperCase() + sentence.slice(1);
console.log(capitalizedSentence); // "First letter capitalized"
```
