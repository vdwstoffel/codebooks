# Useful Libraries

This page provides an overview of several useful libraries that can help you build React applications more efficiently.

## React Hot Toast

[React Hot Toast](https://react-hot-toast.com/) is a lightweight and easy-to-use library for adding toast notifications to your application.

```bash
npm install react-hot-toast
```

### Basic Usage

To use React Hot Toast, import the `toast` function and the `Toaster` component. The `Toaster` component renders the toasts, and the `toast` function is used to trigger them.

```jsx
import toast, { Toaster } from "react-hot-toast";

const notify = () => toast.success("Here is your toast.");

export default function HotToast() {
  return (
    <div>
      <button onClick={notify}>Make me a toast</button>
      <Toaster />
    </div>
  );
}
```

## React Hook Form

[React Hook Form](https://react-hook-form.com/) is a performant, flexible, and extensible library for managing forms in React. It leverages hooks to manage form state and validation, which results in less code and better performance.

```bash
npm install react-hook-form
```

### Basic Usage

The `useForm` hook provides methods for registering inputs, handling submissions, and accessing form state and errors.

```jsx
import { useForm } from "react-hook-form";

export default function App() {
  const {
    register,
    handleSubmit,
    formState: { errors },
  } = useForm();

  const onSubmit = (data) => console.log(data);

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input defaultValue="test" {...register("example")} />
      <input {...register("exampleRequired", { required: true })} />
      {errors.exampleRequired && <span>This field is required</span>}
      <input type="submit" />
    </form>
  );
}
```

## Leaflet

[Leaflet](https://leafletjs.com/) is a leading open-source JavaScript library for mobile-friendly interactive maps. [React-Leaflet](https://react-leaflet.js.org/) provides React components for Leaflet maps.

```bash
npm install leaflet react-leaflet
```

### Basic Setup

To get started, you need to import the Leaflet CSS and then use the `MapContainer`, `TileLayer`, and `Marker` components.

```jsx
import { MapContainer, TileLayer, Marker, Popup } from "react-leaflet";
import "leaflet/dist/leaflet.css";

export default function LeafletGettingStarted() {
  return (
    <MapContainer center={[51.505, -0.09]} zoom={13} style={{ height: "400px" }}>
      <TileLayer
        url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
        attribution='&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
      />
      <Marker position={[51.505, -0.09]}>
        <Popup>A pretty CSS3 popup. <br /> Easily customizable.</Popup>
      </Marker>
    </MapContainer>
  );
}
```

### Re-Center map to new location

```jsx
import { useState } from "react";
import { MapContainer, TileLayer, Marker, Popup, useMap } from "react-leaflet";

import "./leaflet.css";

export default function LeafletRecenterMap() {
  const [position, setPosition] = useState([51, 3]);
  return (
    <>
      <div>
        <button onClick={() => setPosition([51, 9])}>Pos 1</button>
        <button onClick={() => setPosition([46, 18])}>Pos 2</button>
      </div>
      <MapContainer
        center={position}
        zoom={13}
        scrollWheelZoom={true}
        style={{ height: "500px", width: "500px" }}
      >
        <TileLayer
          attribution='&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
          url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
        />
        <Marker position={position}>
          <Popup>
            A pretty CSS3 popup. <br /> Easily customizable.
          </Popup>
        </Marker>
        <ChangeCenter position={position} />
      </MapContainer>
    </>
  );
}

function ChangeCenter({ position }) {
  const map = useMap();
  map.setView(position);
  return null;
}
```

### Add click event to map

```jsx
import {
  MapContainer,
  TileLayer,
  Marker,
  Popup,
  useMapEvents,
} from "react-leaflet";

import "./leaflet.css";
import { useState } from "react";

export default function MapClickEvent() {
  const [selectedLocation, setSelectedLocation] = useState({
    lat: null,
    lon: null,
  });

  function ClickEvent() {
    useMapEvents({
      click: (e) => {
        console.log(e.latlng);
        setSelectedLocation({ lat: e.latlng.lat, lng: e.latlng.lng });
      },
    });
    return null;
  }

  return (
    <>
      <h1>
        Clicked Location: {selectedLocation.lat}, {selectedLocation.lng}
      </h1>
      <MapContainer
        center={[51.505, -0.09]}
        zoom={13}
        scrollWheelZoom={true}
        style={{ height: "500px", width: "500px" }}
      >
        <TileLayer
          attribution='&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
          url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
        />
        <Marker position={[51.505, -0.09]}>
          <Popup>
            A pretty CSS3 popup. <br /> Easily customizable.
          </Popup>
        </Marker>
        <ClickEvent />
      </MapContainer>
    </>
  );
}
```

## date-fns

[date-fns](httpshttps://date-fns.org/) is a modern JavaScript date utility library. It provides a comprehensive set of functions for manipulating and formatting dates.

```bash
npm install date-fns
```

### Basic Usage

Here are a few examples of how you can use `date-fns` to format and manipulate dates:

```jsx
import { format, addDays, differenceInDays } from 'date-fns';

const today = new Date();

// Format a date
const formattedDate = format(today, 'MMMM do, yyyy'); // "July 21st, 2025"

// Add 5 days to a date
const futureDate = addDays(today, 5);
const formattedFutureDate = format(futureDate, 'MMMM do, yyyy'); // "July 26th, 2025"

// Calculate the difference between two dates
const date1 = new Date('2025-07-21');
const date2 = new Date('2025-07-26');
const dayDifference = differenceInDays(date2, date1); // 5
```

## React Select

[React Select](https://react-select.com/) is a flexible and beautiful Select Input control for ReactJS with multiselect, autocomplete, and ajax support.

```bash
npm install react-select
```

### Basic Usage

Here is a basic example of how to use React Select with a simple list of options.

```jsx
import Select from 'react-select';

const options = [
  { value: 'chocolate', label: 'Chocolate' },
  { value: 'strawberry', label: 'Strawberry' },
  { value: 'vanilla', label: 'Vanilla' },
];

export default function App() {
  return <Select options={options} />;
}
```
