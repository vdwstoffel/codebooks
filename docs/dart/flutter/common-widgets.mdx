# Common Widgets

## Layout

### Scaffold
Implements the basic material design visual layout structure.
```dart
Scaffold(
  appBar: AppBar(title: Text('My App')),
  body: Center(child: Text('Hello')),
);
```

### Row
A widget that displays its children in a horizontal array.
```dart
Row(
  children: <Widget>[
    Text('Hello'),
    Text('World'),
  ],
);
```

### Column
A widget that displays its children in a vertical array.
```dart
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: <Widget>[
    Text('Hello'),
    Text('World'),
  ],
);
```

### Stack
A widget that positions its children relative to the edges of its box.
```dart
Stack(
  children: <Widget>[
    Container(color: Colors.blue),
    Center(child: Text('Hello')),
  ],
);
```

### Container
A convenience widget that combines common painting, positioning, and sizing widgets.
```dart
Container(
  width: double.infinity,
  height: double.infinity,
  color: Colors.blue,
  padding: EdgeInsets.all(16.0),
  child: Text('Hello'),
);
```

### SizedBox
A box with a specified size. It can be used to add space between widgets.
```dart
SizedBox(
  width: 200.0,
  height: 300.0,
  child: const Card(child: Text('Hello World')),
)
```

### SingleChildScrollView
A box that can be scrolled. This widget is useful when you have a single widget that is too large to fit on the screen.
```dart
SingleChildScrollView(
  child: Column(
    children: List.generate(
      20,
      (index) => ListTile(
        title: Text('Item $index'),
      ),
    ),
  ),
)
```

## Positioning

### Center
A widget that centers its child within itself.
```dart
Center(
  child: Text('Hello'),
);
```

### Padding
A widget that insets its child by a given padding.
```dart
Padding(
  padding: EdgeInsets.all(16.0),
  child: Text('Hello'),
);
```

### Expanded
A widget that expands a child of a `Row`, `Column`, or `Flex` so that the child fills the available space.
```dart
Row(
  children: <Widget>[
    Expanded(
      child: Container(
        color: Colors.blue,
        height: 100,
      ),
    ),
    Container(
      color: Colors.red,
      width: 100,
      height: 100,
    ),
  ],
);
```

## Structures

### Button
A material design button. Ex, `TextButton`, `ElevatedButton`, `OutlinedButton`
```dart
ElevatedButton(
  onPressed: () {},
  child: Text('Click Me'),
);
```
<p>Outlined Button</p>
```dart
OutlinedButton(
  onPressed: () {},
  child: Text('Click Me'),
  style: OutlinedButton.styleFrom(
    side: BorderSide(color: Colors.blue),
  ),
);
```
<p>Text Button</p>
```dart
TextButton(
  onPressed: () {},
  child: Text('Click Me'),
  style: TextButton.styleFrom(
    foregroundColor: Colors.red,
  ),
);
```
<p>Elevated Button with custom style</p>
```dart
ElevatedButton(
  onPressed: () {},
  child: Text('Click Me'),
  style: ElevatedButton.styleFrom(
    backgroundColor: Colors.green,
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(20),
    ),
  ),
);
```

### Text
A run of text with a single style.
```dart
Text('Hello, world!');
```

## Styles

### TextStyle
An immutable style describing how to format and paint text.
```dart
Text(
  'Hello',
  style: TextStyle(fontSize: 24.0),
);
```

### Color
An immutable 32-bit color value in ARGB format.
```dart
Container(
  color: Colors.blue,
);
```
