# Dart Lists

In Dart, a `List` is an ordered collection of objects. It is the equivalent of an array in other programming languages.

## Creating a List

You can create a list using a list literal or the `List` constructor.

```dart
// Using a list literal
var list = [1, 2, 3];

// A list of strings
var stringList = List<String>.of(['one', 'two', 'three']);

// Using the List constructor
var list2 = List<int>.filled(3, 0); // [0, 0, 0]
```

## Accessing Elements

You can access list elements by their index. The index is zero-based.

```dart
var list = ['a', 'b', 'c'];

list[0]; // 'a'
list[1]; // 'b'
```

## Common List Properties and Methods

Here are some common properties and methods for lists:

```dart
var list = [10, 20, 30];

// Get the length of the list
list.length; // 3

// Add an element to the end
list.add(40);
// list is now [10, 20, 30, 40]

// Remove an element
list.remove(20);
// list is now [10, 30, 40]

// Get the index of an element
list.indexOf(30); // 1

// Check if the list is empty
list.isEmpty; // false

// Check if the list is not empty
list.isNotEmpty; // true
```

## Iterating Over a List

You can iterate over a list using a `for` loop or the `forEach` method.

```dart
var list = ['one', 'two', 'three'];

for (var item in list) {
  print(item);
}

list.forEach((item) => print(item));
```

## Higher-Order Methods

Lists also support powerful higher-order methods like `map`, `where`, and `reduce`.

```dart
var numbers = [1, 2, 3, 4];

// Create a new list with each element doubled
var doubled = numbers.map((n) => n * 2).toList(); // [2, 4, 6, 8]

// Create a new list with only the even numbers
var evens = numbers.where((n) => n.isEven).toList(); // [2, 4]

// Calculate the sum of the list
var sum = numbers.reduce((value, element) => value + element); // 10
```
