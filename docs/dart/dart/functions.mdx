# Functions

Functions are the building blocks of readable, maintainable, and reusable code. A function is a set of statements to perform a specific task.

## Function with no return value

This type of function is often called a "void" function. It performs an action but doesn't return a value.

```dart
void sayHello() {
  print('Hello, Dart!');
}

void main() {
  sayHello(); // Hello, Dart!
}
```

## Function with an Integer Return
This function takes two integers as input and returns their sum.

```dart
int add(int a, int b) {
  return a + b;
}

void main() {
  int sum = add(5, 3);
  print(sum); // 8
}
```

## Function with a String Return
This function returns a simple greeting message as a string.

```dart
String getGreeting() {
  return 'Hello, from a function!';
}

void main() {
  String greeting = getGreeting();
  print(greeting); // Hello, from a function!
}
```

## Function with Arguments

Functions can take arguments, which are values you pass into the function.

```dart
void greetUser(String name) {
  print('Hello, $name!');
}

void main() {
  greetUser('Alice'); // Hello, Alice!
}
```

## Named Arguments

Dart allows you to use named arguments to make your function calls more readable. You can also use the `required` keyword to indicate that a named argument is mandatory.

```dart
void printUserInfo({required String name, required int age}) {
  print('Name: $name, Age: $age');
}

void main() {
  printUserInfo(name: 'Bob', age: 42); // Name: Bob, Age: 42
}
```

## Default Arguments

You can assign default values to function parameters. If a value is not provided for a parameter with a default value, the default value is used.

```dart
void sendGreeting(String name, {String greeting = 'Hello'}) {
  print('$greeting, $name!');
}

void main() {
  sendGreeting('Alice'); // Hello, Alice!
  sendGreeting('Bob', greeting: 'Hi'); // Hi, Bob!
}
```
