## Compile and Run Java App

```java title="HelloWorld.java"
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```bash
javac HelloWorld.java
java HelloWorld
```



## Type casting

```java
int a = 10;
int b = 4;

double c = (double) a / b; // 2.5
```
