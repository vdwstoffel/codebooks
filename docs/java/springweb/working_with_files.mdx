# Working with files

## Creating and sending a file

<Tabs>
<TabItem value="FileWriter" >

```java
import java.io.File;
import java.io.FileWriter;
import java.io.IOException;

public class FileService {

    public File createFile() throws IOException {
        File file = new File("output.txt");
        try (FileWriter writer = new FileWriter(file)) {
            writer.write("This is example content written to the file.");
        }
        return file;
    }
}
```

</TabItem>
<TabItem value="Controller" >

```java
import org.springframework.core.io.FileSystemResource;
import org.springframework.http.HttpHeaders;

import java.io.File;
import java.io.IOException;

@RestController
@RequestMapping("/file")
public class FileController {

    private final FileService fileService = new FileService();

    @GetMapping
    public ResponseEntity<FileSystemResource> getFile() {
        try {
            File file = fileService.createFile();
            FileSystemResource resource = new FileSystemResource(file);

            HttpHeaders headers = new HttpHeaders();
            headers.add(HttpHeaders.CONTENT_DISPOSITION, "attachment; filename=" + file.getName());

            return new ResponseEntity<>(resource, headers, HttpStatus.OK);
        } catch (IOException e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
        }
    }
}
```

</TabItem>

## Uploading and Parsing a Text File

This example demonstrates how to upload a text file to a Spring Boot controller and parse its content as a string.

<Tabs>
<TabItem value="Controller" label="Controller">

```java
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;
import org.springframework.web.multipart.MultipartFile;

import java.io.IOException;

@RestController
@RequestMapping("/upload")
public class FileUploadController {

    @PostMapping("/text")
    public ResponseEntity<String> uploadTextFile(@RequestParam("file") MultipartFile file) {
        if (file.isEmpty()) {
            return ResponseEntity.status(HttpStatus.BAD_REQUEST).body("Please select a file to upload.");
        }

        try {
            // Get the content of the file as a string
            String content = new String(file.getBytes());

            // Here you can process the content of the file
            System.out.println("File content: " + content);

            return ResponseEntity.ok("File uploaded and content parsed successfully.");
        } catch (IOException e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body("Failed to read file content.");
        }
    }
}
```

</TabItem>
<TabItem value="Usage" label="Usage">

To upload a file, you can use a tool like `curl`:

```bash
curl -X POST -F "file=@/path/to/your/file.txt" http://localhost:8080/upload/text
```

Replace `/path/to/your/file.txt` with the actual path to your text file.
