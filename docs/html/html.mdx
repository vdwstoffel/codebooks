## Input validation

```html
<body>
    <label>Name</label>
    <input type="text" class="userInput" minlength="3" required />
    <label>email</label>
    <input type="email" required class="userInput" />
</body>
```

```css
.userInput:user-valid {
    outline: 1px solid greenyellow;
}

.userInput:user-invalid {
    outline: 1px solid chocolate
}
```

<div>
    <label>Name</label>
    <input type="text" class="userInput" minlength="3" required />
    <label>email</label>
    <input type="email" required class="userInput" />
</div>

});
