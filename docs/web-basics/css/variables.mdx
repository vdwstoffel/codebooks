# CSS Variables

CSS variables (also known as CSS custom properties) allow you to store values that can be reused throughout your stylesheet. They make your CSS more maintainable and easier to update.

## Basic Syntax

CSS variables are defined using the `--` prefix and accessed using the `var()` function:

```css title="styles.css"
:root {
  // highlight-next-line
  --primary-color: #007bff;
  // highlight-next-line
  --font-size: 16px;
}

.button {
  // highlight-next-line
  background-color: var(--primary-color);
  // highlight-next-line
  font-size: var(--font-size);
}
```

## Scope and Inheritance

Variables defined in `:root` are global, but you can also define them locally:

```css title="scoped-variables.css"
:root {
  --global-color: blue;
}

.component {
  // highlight-start
  --local-color: red;
  --component-spacing: 15px;
  // highlight-end
  
  color: var(--local-color);
  padding: var(--component-spacing);
}

.nested {
  // highlight-next-line
  color: var(--local-color); /* Inherits from parent */
  // highlight-next-line
  background: var(--global-color); /* Uses global variable */
}
```

## Fallback Values

You can provide fallback values in case a variable is not defined:

```css title="fallbacks.css"
.element {
  // highlight-next-line
  color: var(--undefined-color, #333); /* Falls back to #333 */
  // highlight-next-line
  font-size: var(--font-size, var(--default-font, 16px)); /* Nested fallbacks */
}
```

## Dynamic Updates with JavaScript

CSS variables can be updated dynamically with JavaScript:

```html title="dynamic-variables.html"
<style>
  :root {
    --theme-color: #007bff;
  }
  
  .dynamic-box {
    background-color: var(--theme-color);
    width: 100px;
    height: 100px;
    transition: background-color 0.3s;
  }
</style>

<div class="dynamic-box"></div>
<button onclick="changeTheme()">Change Theme</button>

<script>
function changeTheme() {
  const colors = ['#007bff', '#28a745', '#dc3545', '#ffc107'];
  const randomColor = colors[Math.floor(Math.random() * colors.length)];
  
  // highlight-next-line
  document.documentElement.style.setProperty('--theme-color', randomColor);
}
</script>
```

## Benefits of CSS Variables

1. **Maintainability**: Change values in one place to update them everywhere
2. **Consistency**: Ensure consistent values across your stylesheet
3. **Dynamic Updates**: Modify values with JavaScript for interactive themes
4. **Inheritance**: Variables cascade down the DOM tree
5. **Fallbacks**: Provide backup values for better browser support

## Common Use Cases

- **Theme colors and branding**
- **Typography scales** (font sizes, line heights)
- **Spacing systems** (margins, paddings)
- **Component variants** (button sizes, card styles)
- **Responsive breakpoints** and layout values
- **Animation durations** and timing functions
