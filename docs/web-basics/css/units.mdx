# CSS Units

| Unit | Description |
| :--- | :--- |
| `px` | **Pixels (absolute):** Represents a single dot on the screen. Good for precise control, but doesn't scale. |
| `em` | **Relative to parent's font-size:** `1em` is equal to the font-size of the parent element. |
| `rem`| **Relative to root font-size:** `1rem` is equal to the font-size of the `<html>` element. Consistent and scalable. |
| `%`  | **Percentage:** Relative to the parent element's value for a given property. |
| `vw` | **Viewport Width:** 1% of the viewport's width. |
| `vh` | **Viewport Height:** 1% of the viewport's height. |
