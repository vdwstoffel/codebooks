# Flexbox

Flexbox is a one-dimensional layout model that allows you to design complex layout structures more easily. It provides a more efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic.

## Core Concepts

The two main components of a flexbox layout are the **flex container** (the parent element) and the **flex items** (the children).

- **Flex Container**: You can turn any element into a flex container by setting its `display` property to `flex` or `inline-flex`.
- **Flex Items**: The direct children of a flex container automatically become flex items.

Flexbox operates along two axes:
- **Main Axis**: This is the primary axis along which flex items are laid out. It is defined by the `flex-direction` property.
- **Cross Axis**: This is the axis perpendicular to the main axis.

## Examples

### Creating a Flex Container

To get started with Flexbox, you need to first create a flex container.

<Tabs>
<TabItem value="html" label="HTML">
```html
<div class="flex-container">
  <div class="flex-item">Item 1</div>
  <div class="flex-item">Item 2</div>
  <div class="flex-item">Item 3</div>
</div>
```
</TabItem>
<TabItem value="css" label="CSS">
```css
.flex-container {
  /* highlight-next-line */
  display: flex;
  padding: 1rem;
  background-color: #f0f0f0;
  gap: 0.5rem;
}

.flex-item {
  background-color: #4caf50;
  color: white;
  padding: 1rem;
  border-radius: 4px;
}
```
</TabItem>
</Tabs>

## justify-content

The `justify-content` property aligns flex items along the main axis.

### flex-start

Items are packed toward the start of the flex-direction.

<BrowserWindow url="http://localhost:3000">

        <div style={{ backgroundColor: '#2196f3', color: 'white', padding: '1rem', borderRadius: '4px' }}>Item 2</div>
        <div style={{ backgroundColor: '#ff9800', color: 'white', padding: '1rem', borderRadius: '4px' }}>Item 3</div>
    </div>
</BrowserWindow>

## align-items

The `align-items` property aligns flex items along the cross axis.

### stretch
