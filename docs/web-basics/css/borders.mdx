# CSS Borders

CSS borders allow you to create a line around any HTML element. You can control the style, width, and color of the border to create various visual effects.

## Key Concepts

| Property          | Description                                                                                              |
| ----------------- | -------------------------------------------------------------------------------------------------------- |
| `border-style`    | Sets the style of the border (e.g., `solid`, `dotted`, `dashed`).                                        |
| `border-width`    | Specifies the thickness of the border.                                                                   |
| `border-color`    | Sets the color of the border.                                                                            |
| `border`          | A shorthand property for setting `border-width`, `border-style`, and `border-color` in one declaration. |
| `border-radius`   | Rounds the corners of the border.                                                                        |

## Examples

### Solid Border

This example shows how to create a simple solid border around a `div` element.

<Tabs>
  <TabItem value="html" label="HTML">
    ```html
    <div class="solid-border">This div has a solid border.</div>
    ```
  </TabItem>
  <TabItem value="css" label="CSS">
    ```css
    .solid-border {
      border: 2px solid #007bff;
      padding: 10px;
    }
    ```
  </TabItem>
</Tabs>

<BrowserWindow>
  <div style={{ border: '2px solid #007bff', padding: '10px' }}>
    This div has a solid border.
  </div>
</BrowserWindow>
