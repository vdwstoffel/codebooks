# CSS Transitions and Transformations

CSS transitions and transformations are powerful features that allow you to create animations and visual effects, making your web pages more dynamic and engaging.

## Transitions

A CSS transition provides a way to control the speed of an animation when changing CSS properties. Instead of having a property change take effect immediately, you can cause the change to occur over a period of time.

### Key Properties

- `transition-property`: Specifies the name of the CSS property to which the transition is applied.
- `transition-duration`: Defines the length of time that a transition takes.
- `transition-timing-function`: Describes how the speed of the transition changes over its duration (e.g., `ease`, `linear`, `ease-in-out`).
- `transition-delay`: Sets a delay before the transition starts.

### Example

In this example, the background color of the box changes smoothly over 0.5 seconds when you hover over it.

<Tabs>
  <TabItem value="html" label="HTML">
    ```html
    <div class="transition-box">
      Hover me
    </div>
    ```
  </TabItem>
  <TabItem value="css" label="CSS">
    ```css
    .transition-box {
      width: 100px;
      height: 100px;
      background-color: #3498db;
      color: white;
      display: flex;
      align-items: center;
      justify-content: center;
      transition: background-color 0.5s ease;
    }

    .transition-box:hover {
      background-color: #e74c3c;
    }
    ```
  </TabItem>
</Tabs>

## Transformations

A CSS transformation allows you to move, rotate, scale, and skew elements in 2D or 3D space. It changes the shape and position of an element without affecting the surrounding elements.

### Key Functions

- `translate()`: Moves an element from its current position.
- `rotate()`: Rotates an element around a point.
- `scale()`: Changes the size of an element.
- `skew()`: Skews an element along the X or Y axis.

### Example

This example shows a box that is scaled up and rotated when you hover over it.

<Tabs>
  <TabItem value="html" label="HTML">
    ```html
    <div class="transform-box">
      Hover me
    </div>
    ```
  </TabItem>
  <TabItem value="css" label="CSS">
    ```css
    .transform-box {
      width: 100px;
      height: 100px;
      background-color: #2ecc71;
      color: white;
      display: flex;
      align-items: center;
      justify-content: center;
      transition: transform 0.2s ease;
    }

    .transform-box:hover {
      transform: scale(1.2) rotate(15deg);
    }
    ```
  </TabItem>
</Tabs>

## Combining Transitions and Transformations

You can combine transitions and transformations to create smooth and complex animations. The transition will apply to the changes made by the transform, resulting in a fluid visual effect.

### Example

In this example, hovering over the box will cause it to smoothly scale up and rotate.

<Tabs>
  <TabItem value="html" label="HTML">
    ```html
    <div class="combined-box">
      Hover me
    </div>
    ```
  </TabItem>
  <TabItem value="css" label="CSS">
    ```css
    .combined-box {
      width: 100px;
      height: 100px;
      background-color: #9b59b6;
      color: white;
      display: flex;
      align-items: center;
      justify-content: center;
      transition: transform 0.4s ease-in-out;
    }

    .combined-box:hover {
      transform: scale(1.3) rotate(-10deg);
    }
    ```
  </TabItem>
</Tabs>
