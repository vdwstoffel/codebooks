# HTML Tables

HTML tables are used to present data in a two-dimensional format, with rows and columns. They are useful for displaying tabular data, such as financial reports, schedules, and statistics.

## Basic Table Structure

A basic HTML table consists of the `<table>` element, which contains one or more `<tr>` (table row), `<th>` (table header), and `<td>` (table data) elements.

<Tabs>
  <TabItem value="html" label="HTML">
    ```html
    <table>
      <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
      </tr>
      <tr>
        <td>Jill</td>
        <td>Smith</td>
        <td>50</td>
      </tr>
      <tr>
        <td>Eve</td>
        <td>Jackson</td>
        <td>94</td>
      </tr>
    </table>
    ```
  </TabItem>
  <TabItem value="css" label="CSS">
    ```css
    table, th, td {
      border: 1px solid black;
      border-collapse: collapse;
      padding: 8px;
    }
    th {
      background-color: #f2f2f2;
    }
    ```
  </TabItem>
</Tabs>

## Table with Colspan

The `colspan` attribute can be used to make a cell span multiple columns.

<Tabs>
  <TabItem value="html" label="HTML">
    ```html
    <table>
      <tr>
        <th>Name</th>
        <td colspan="2">Telephone</td>
      </tr>
      <tr>
        <td>Bill Gates</td>
        <td>555 77 854</td>
        <td>555 77 855</td>
      </tr>
    </table>
    ```
  </TabItem>
  <TabItem value="css" label="CSS">
    ```css
    table, th, td {
      border: 1px solid black;
      border-collapse: collapse;
      padding: 8px;
    }
    th {
      background-color: #f2f2f2;
    }
    ```
  </TabItem>
</Tabs>
